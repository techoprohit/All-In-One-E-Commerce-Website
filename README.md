## Description
All in One is a static e-commerce website built using HTML, CSS, and JavaScript. It provides a complete shopping experience with features like product browsing, shopping cart, checkout, and user login. The website is designed to be responsive and user-friendly, offering a variety of products across different categories.

## DEPLOYMENT LINK:
(https://techoprohit.github.io/All-In-One-E-Commerce-Website)

## Features
- **Home Page**: Welcome banner, category browsing, and featured products.
- **Product Listing**: Browse products by category and price filters.
- **Product Details**: Detailed view of individual products with image gallery.
- **Shopping Cart**: Add, remove, and update quantities of items in the cart.
- **Checkout**: Form for placing orders with payment method selection.
- **User Login**: Simple login form that stores credentials in local storage.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Local Storage**: Cart data persists across browser sessions.

## Technologies Used
- **HTML5**: Structure and content of the website.
- **CSS3**: Styling, including gradients, transitions, and responsive design.
- **JavaScript (ES6)**: Functionality for cart management, form handling, and dynamic content.
- **Local Storage**: For storing cart items and user login details.

## How to Run
1. Clone or download the project files to your local machine.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, etc.).
3. Navigate through the website using the menu links.

## Usage
- **Browsing Products**: Visit the Products page to see all available items. Use filters to narrow down by category or price.
- **Adding to Cart**: Click "Add to Cart" on any product to add it to your shopping cart.
- **Managing Cart**: Go to the Cart page to view, update quantities, or remove items.
- **Checkout**: Proceed to checkout to fill in shipping details and select payment method.
- **Login**: Use the login form to simulate user authentication (credentials are stored locally).
